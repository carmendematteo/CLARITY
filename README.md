A stock/crypto guide tool.

Created by Carmen DeMatteo (me) for Tyrant Investing (my investment partnership). Built with the help of Claud Sonnet 4.6, and Gemini 3.1 Pro. Design inspired by iOS.

CLARITY uses equations to cut through market noise and identify the true estimated value of assets. Requiring only honest, high quality data, the tool provides a baseline for the health of an asset and accounts for human psychology, game theory, and chaos theory.
